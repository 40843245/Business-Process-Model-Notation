# quickstart guide
## Elements
Elements in BPMN is a small unit of an enitity. It behaves like entities in ERD.

It includes 

+ Flow objects
+ Connecting objects
+ Swim lanes
  
## Flow objects
+ events
+ activities
+ gateways

### Connecting objects
+ sequence flow
+ message flow
+ association

## Swim lanes

+ lane
+ dark pool
